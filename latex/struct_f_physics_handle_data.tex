\doxysubsection{Detailed Description}
VR Physics Handle data, holds all of this classes functionality variables. Defaults values are tested with 1kg grabbable assets. N\+O\+TE\+: Values may need to be higher to have effect on lighter components less than 1KG and heavier may need weaker values to prevent collision issues. N\+O\+TE\+: Mainly used to switch between different functionality using the Update\+Joint\+Values function.

\doxysubsection{Detailed Description}
Box collision component. Used to enable soft collision only from the hand class for any interactables that can be picked up. N\+O\+TE\+: Use-\/full for preventing things being forces through collisions due to the hand\+Skel having infinite mass and force as it does not simulate physics.

\doxysubsection{Detailed Description}
A simple version of a slidable static mesh actor that can be used for things that do not need collisions, like sliders on panel of electronics, or things like inserting a floppy disc etc. N\+O\+TE\+: Will slide in its relative selected axis to its parent component/actor.

Detailed Description

To adjust the buttons update distance make the intractable sphere collider larger. NOTE: Created a blueprint class from this class in editor for use in blueprint based actors. Also this button updates every frame, to improve this only update this button when overlapping with a shape collider in the actor blueprint/class.
